# Trufla_Task
To Run this :-

clone the repo

Go to the Trufla Task folder

                   cd  Trufla Task
activate virtual env

                   Trufla/Scripts/activate


install the requirements

                  pip install -r requirements
Run the files

                  python solution.py
                  python solution2.py

Output will be inside parsing_result folder

                  cd parsing_result
